# 📊 PhonePe Transaction Analysis Dashboard (Power BI)

## 📌 Project Overview

This project analyzes **PhonePe digital payment data** using **Microsoft Power BI** to uncover insights about transaction trends, payment categories, and regional usage across India.

The dashboard provides interactive visualizations to help understand:

* Transaction volume and value
* State-wise and district-wise payment trends
* Category-wise digital payment distribution
* Growth of digital payments over time

The goal of this project is to demonstrate **data analysis, visualization, and business insight generation using Power BI**.

## 📊 Dashboard Features

The Power BI dashboard includes the following insights:

### 1️⃣ Transaction Overview

* Total transaction amount
* Total transaction count
* Year-wise digital payment growth

### 2️⃣ State-wise Analysis

* Transactions across different Indian states
* Comparison of transaction volume and value

### 3️⃣ Category Analysis

Breakdown of payments such as:

* Recharge & Bills
* Merchant Payments
* Financial Services
* Insurance
* Loans

### 4️⃣ Trend Analysis

* Monthly transaction trends
* Growth of digital payments over time

---

## 📈 Key Insights

* Digital payments show significant growth across Indian states.
* Merchant payments contribute a major share of transactions.
* Some states dominate in both transaction count and transaction value.
* Increasing adoption of digital platforms for daily payments.
